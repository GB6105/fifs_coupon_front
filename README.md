# 미션 - 선착순 쿠폰 발급 프로그램

해당 프로젝트는 오픈 미션- 선착순 쿠폰 발급 프로그램을 완성한 뒤,   
실제 서버에서 동작하는 것을 확인하기 위해 추가한 프로젝트입니다.

## 실습 예제

<img width="1140" height="912" alt="image" src="https://github.com/user-attachments/assets/956d3f07-e5da-492e-9028-4c2ce50ab3d1" />


첫 화면에서 이메일을 입력하고 버튼을 누르면 쿠폰 목록 페이지로 이동합니다.

<img width="1107" height="907" alt="image" src="https://github.com/user-attachments/assets/33b8f7e6-4717-4ca6-9867-b0ab68d68768" />

원하는 쿠폰 버튼을 누르면 발급 페이지로 이동합니다.

<img width="1119" height="917" alt="image" src="https://github.com/user-attachments/assets/3edca618-167e-4c04-bad0-e97e92fc5242" />

발급 버튼을 눌러서 신청할 수 있습니다.

<img width="1115" height="913" alt="image" src="https://github.com/user-attachments/assets/8debe3d5-71dd-4001-8141-d8cb94c4738f" />

발급에 성공하면 수량이 감소하고 목록에 반영됩니다.
기존에 발급한 기록이 있거나 수량이 부족하면 발급에 실패합니다.
